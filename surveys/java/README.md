### Instructions to Run the Machine Learning Script


 * Install Python

```
sudo apt-get update
sudo apt-get install python
```

 * Install Pip

```
sudo apt-get install python-pip
```

 * Install python-tk

```
sudo apt-get install python-tk
```

 * Run the following commands:

```
cd machine-learning
pip install -r requirements.txt
python script.py
```
